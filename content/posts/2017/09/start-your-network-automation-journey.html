He continued with…

I’m looking forward to see the other webinars as I’m starting to bring network automation to our department.

This is the sequence I would use to go through them:

  • Network Automation 101 is another high-level webinar focusing more on network automation concepts, goals and caveats.
  • Network Automation Tools is an overview of open-source tools you can use.
  • Ansible for Networking Engineer is the real deep-dive into Ansible and its use in network automation.
  • Network Automation Use Cases is a collection of real-life use cases, from high-level discussions (how do you automate IPv6 deployments) to hands-on examples (how do you deploy a data center fabric).
  • You might also want to know how NETCONF works.
  • Also, make sure you listen to Software Gone Wild podcast – you’ll find tons of useful information there.

I guess you already know that you can get all these webinars with ipSpace.net subscription.
