--- url: /2006/11/log-user-privilege-level-changes/ title: "Log user privilege level changes" date: "2006-11-24T19:13:00.000+01:00" tags: [ syslog,access control,logging ] --- The logging userinfo configuration command introduced in IOS release 12.3T (integrated into 12.4) starts logging of all changes in user privilege levels (as requested by enable or disable commands).
Sample printout:
fw>enable
Password:
03:00:50: %SYS-5-PRIV_AUTH_PASS: Privilege level set to 15 by unknown on console
fw#disable
fw>
03:00:52: %SYS-5-PRIV_AUTH_PASS: Privilege level set to 1 by unknown on console
