--- date: 2006-10-25T09:20:00.000+02:00 tags: - HTTP - network management - web title: Use command aliases to simplify Cisco IOS web server URLs url: /2006/10/use-command-aliases-to-simplify-cisco/ --- Cisco IOS has the ability to define command aliases - short words that substitute a whole exec-mode or configuration command. These aliases can also be used in command execution URLs. For example, if you define alias exec showdefault show ip route 0.0.0.0 0.0.0.0, you can view the status of the default route with the http://router/exec/showdefault/CR URL.
