--- url: /2008/07/arp-table-with-logical-and-physical.html title: "ARP table with logical and physical interfaces" date: "2008-07-30T07:17:00.001+02:00" tags: [ Tcl,ARP,command line interface ] --- In a layer-3 switching environment, the ARP table displayed with the show arp command lists the logical (L3) interfaces, for example the VLAN or BVI interface. This Tcl script displays the logical as well as physical interface associated with each IP/MAC address.
