--- title: "Disable optional IOS features on high CPU load" date: "2008-06-23T06:35:00.000+02:00" tags: [ network management,configuration,EEM ] --- One of my readers has submitted an interesting EEM applet in a comment to the Generate SNMP trap on high CPU load post. The applet monitors the CPU load (using SNMP variable from the CISCO-PROCESS-MIB) and disables WCCP when the 1-minute average load exceeds 75%. You can change the thresholds or disable/enable other IOS features by modifying the applet's source code.
