_netlab_ release 26.06 adds OSPFv3 support on FortiOS (by [@a-v-popov](https://github.com/a-v-popov)) and MPLS/VPN support on SR Linux. We also ensured the installation scripts work on Ubuntu 26.04 (everything else was OK) and updated the installed Vagrant version to 2.4.9 (we're not using new Vagrant features; you don't have to upgrade it in an existing installation).

Other than that, we added a few [improvements](https://netlab.tools/release/26.06/#new-functionality) and squashed a number of [bugs](https://netlab.tools/release/26.06/#bug-fixes).

### Upgrading or Starting from Scratch?

* To upgrade your *netlab* installation, execute `pip3 install --upgrade networklab`.
* New to *netlab*? Start with the [Getting Started document](https://netlab.tools/tutorials/) and the [installation guide](https://netlab.tools/install/).
