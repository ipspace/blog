To understand all the details of the previous debugging printout, you have to know the exact contents of EIGRP topology table of router Fred. You can compute the topology table by hand or set up the four-router lab and print out the topology table on Fred.

Solution

Build a four router lab from one router with four serial links (Barney) and three routers with at least one serial link (Fred, Wilma and Betty).

Connect the routers using DTE - DCE cables as follows:

  DCE side DTE side
#1 Barney Serial 0 Fred Serial 0
#2 Barney Serial 1 Betty Serial 0
#3 Barney Serial 2 Wilma Serial 0

Load router configurations on the routers. Download the configurations by clicking on the routers in the following picture:

Verify that all the serial interfaces are up and examine the EIGRP topology table on router Fred.
