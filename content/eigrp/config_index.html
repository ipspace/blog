This document contains the router configurations used in the DUAL examples in the EIGRP book

Router requirements

Router name Hardware configuration
Barney at least 3 serial ports
Fred one serial port
Wilma one serial port
Betty one serial port

Cabling requirements

Cable type Quantity
Serial cable - DTE (X.21,RS-232 or V.35) 3
Serial cable - DCE (same type as above) 3

Connect the routers using DTE - DCE cables as follows:

  DCE side DTE side
#1 Barney Serial 0 Fred Serial 0
#2 Barney Serial 1 Betty Serial 0
#3 Barney Serial 2 Wilma Serial 0
