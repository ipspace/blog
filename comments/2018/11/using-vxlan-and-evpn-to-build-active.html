  4. Michael67 27 November 2018 22:24
    Let´s talk about vendor limitations, e.g. HPE 5950 switch. You can configure distributed gateways with IP and IPv6. You can also advertise local connected /32 IP addresses into your routing protocol with command "arp route-direct advertise". But there´s no "nd route-direct advertise" available. So you are limited to IPv4 active-active DCI. This is what you learn if you have the switch locally and HPE says "no" after you raised an enhancement request to add this feature.
