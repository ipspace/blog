      All Nexus switches with hw sensor capabilities will automatically stream telemetry to Tetration when added to the Tetration inventory. To see if your Nexus is in the Tetration inventory, query the Tetration API:
      GET /openapi/v1/switches

      If the switch is in the inventory but not streaming, it might be administratively disabled. Check with:
      GET /openapi/v1/switches/{serial}

      "datapath_disabled" should be false
