    Another approach I've used is to store Cisco ACI API data ( i.e. Ansible URI call for faults extract ) in the .html file format ( filtered appropriately ) and send a batch of them at specific time of day to my email with Cron job. Then I just set a rule on my email to easily find it. Because it's an .html I can just double click on it and it pops up the content in web browser.
