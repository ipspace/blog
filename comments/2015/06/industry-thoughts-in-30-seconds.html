      It's mostly ease-of-use. While it's theoretically possible to implement whatever information distribution mechanism with SNMP, doing so is exceedingly hard when compared to something like REST API with JSON-formatted data.

      It's almost like asking "what's the benefit of this new SSD thing over the tapes we've been using for the last 50 years"... after all, they all get the job done ;)
