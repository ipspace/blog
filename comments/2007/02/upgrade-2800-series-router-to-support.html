    After upgrading my 2811 with the new ROMMON, I was still having problems like this:

    rommon 4 > dir usbflash0:
    program load complete, entry point: 0x8000f000, size: 0x3d240

    The rommon could see the usbflash0: device, but didn't like the filesystem.

    I fixed it by blowing away all of the partitions on the USB disk (a 128MB relic) and creating one large partition using fdisk on linux. Set they type to 6 (FAT 16). Then formatted the partition with 'sudo mkdosfs /dev/sdb1'

    Now the ROMMON is perfectly happy with the filesystem:

    System image file is "usbflash0:c2800nm-advipservicesk9-mz.151-4.M5.bin"

  41. Anonymous 22 January 2017 20:03
    I had the same troubles as everyone above. I was trying to install a new ios, but my current CF did not have enough memory. I bought a new CF off ebay. I used a card reader on my PC and copied the new IOS onto the newly bought CF (was used on ebay). I placed in my router, but would only boot in rommon mode. My router already had the latest rommon version as described above. Router would not recognize my flash. I tried re-formatting on my pc (even though the was already configured for FAT - also tried FAT32), but still would not work. I tried a usb flash, but same thing until I used a really old usb (udisk 4GB), but even though dir usbflash: showed me my ios file, I still could not boot off of it - was getting same errors as above.

    Solution was I put the original flash/ios back in, turned on the router. I then used the router to re-format my usb while in enable mode. I then copied the new ios onto this usb using card reader on PC. I placed back into router and then with the router still on, I ejected the old flash/ios (making sure the light next to CF was off) and then placed the new flash in and then from enable mode formatted the new flash. I then copied the new ios from the usb to flash. I changed my configuration register back to 0x2102 from 0x2101 (since I tried using the "boot system" command as described above). I reloaded my router and everything is good now.
