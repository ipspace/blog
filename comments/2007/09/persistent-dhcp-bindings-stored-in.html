  4. Ivan Pepelnjak 05 September 2007 12:38

    If you have a “traditional” flash file system, the files never disappear, they are just marked as deleted (and you have to squeeze it manually). As you have to store the DHCP bindings in a file whenever something changes (after a short delay, of course), you could have a lot of old versions left in the flash after a while … not something I would want to have in a production network.
