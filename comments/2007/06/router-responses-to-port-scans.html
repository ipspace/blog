    Nmap has also a very useful command line option, which helps to understand, which packets the tool has sent/received. It's called --packet-trace. It shows you which packets, nmap is sending and which ones it has received, simmilar to the debug output.

    Example:

    nmap -sS -n -P0 -p 80 --packet-trace www.heise.de
