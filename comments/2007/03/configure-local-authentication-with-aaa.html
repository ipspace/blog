    If you use the default list you don't have to make any changes to the Console or VTY lines. They will automatically use the default aaa list. For example:

    aaa new-model
    aaa authentication login default group local

    You can also use this in your corporate network:

    aaa new-model
    aaa authentication login default group TACACS+ local

    No passwords on lines, no authentication commands on lines. Nice and clean.
