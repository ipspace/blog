    [alias]
wdiff = diff --word-diff --word-diff-regex='\\w+'
  4. Joseph W. 21 August 2025 08:13

    Nice! Unfortunately this ignores non-word characters in the diff (see https://git-scm.com/docs/git-diff#Documentation/git-diff.txt---word-diff-regexregex). You can fix it using something like:

    gitdf --word-diff-regex=&#39;\w+|\s+&#39;
