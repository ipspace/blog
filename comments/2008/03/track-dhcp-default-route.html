  4. Anonymous 30 August 2009 23:49
    I fought this trying to use a static route. I got it working with a floating route in the interface.

    interface FastEthernet0
    ip dhcp client route track 101
    ip address dhcp client-id FastEthernet0
    ip virtual-reassembly
    ....

    interface FastEthernet1
    ip dhcp client route track 102
    ip address dhcp client-id FastEthernet1
    ip virtual-reassembly
    ....
