  2. t 12 January 2011 21:53
    There's a major problem? Let's say you have a link with a backup GRE tunnel. Link is 1500 bytes so MTU is 1500. Then there is a link failure. Traffic now goes over the backup GRE tunnel. However, now the MTU is smaller. The hosts have already done Path MTU Discovery at 1500, so now all traffic will necessarily drop over the backup link, since the router can no longer fragment.
