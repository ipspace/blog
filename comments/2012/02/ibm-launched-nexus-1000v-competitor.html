    IEEE 802.1Qbg support is in the IBM Virtual Fabric 10G Switch Module (for BladeCenter H) starting with the 6.9.1 firmware. Check the Application Guide (Chapter 16: Edge Virtual Bridging) at http://www-01.ibm.com/support/docview.wss?uid=isg3T7000496. VMReady 4.0 is this new Qbg based implementation.

    Qbg Support in KVM is achieved with macvtap (e.g. in RHEL >= 6.1) and can be used with the IBM Virtual Fabric 10G Switch Module.

    P.S. The IBM Virtual Fabric Mode with Emulex 10GbE VFA (CNA) uses Q-in-Q techniques (check the Virtual Fabric Academy website at http://www.virtualfabric.net).
