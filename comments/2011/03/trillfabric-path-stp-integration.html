      If RBridge works on edge physical interfaces, then the BPDUs should not be propagated between ports. If it works on VLAN interfaces, then BPDUs might get propagated.

      In any case, there might be corner cases where BPDUs are propagated between edge ports of the same RBridge. They should never be propagated across TRILL core.
