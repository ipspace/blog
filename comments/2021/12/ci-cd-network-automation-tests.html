    It is not just performance testing that requires a physical lab. There are a lot of platform specific features. Those functional behaviors cannot be tested in a simulator. Virtual routers are far from being the same as physical routers. In most of the simulators, you can only test platform independent features. But do you have a clear understanding of which features are platform dependent? Sometimes it is not that easy to find out... There are also potential software only features that are removed in a hardware assisted architecture. So it is not just a one way problem.
