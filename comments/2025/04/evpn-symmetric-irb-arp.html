    I've used VyOS anycast gateway with symmetric IRB.

    I used multiple VXLAN interfaces, one per L2 VNI and per L3 VNI.

    Each L2 bridge is assigned the same anycast IP and MAC.

    I needed to set "enable-arp-accept" under the IP options. That allows the ARP replies from hosts to be entered into the bridge neighbor table and get a MAC/IP route advertised.

    sudo sysctl -w net.ipv4.conf.br9008.arp_accept=1
