    EOS uses the underlying getaddrinfo function to resolve hosts, which sorts the results using rules in RFC3484. https://man7.org/linux/man-pages/man3/getaddrinfo.3.html

    You'll see the same order by calling getaddrinfo from python e.g. python -c 'import socket ; print( socket.getaddrinfo("b", None))'
